# Frontend

A modern React-based visual pipeline builder for creating and managing data processing workflows. Built with React Flow, Material-UI, and Vite.

## Features

### Visual Pipeline Editor
- **Drag-and-Drop Interface**: Intuitive node-based workflow builder
- **Real-time Validation**: Instant feedback on connections and configurations
- **Auto-Layout**: Smart node positioning and edge routing
- **Zoom & Pan**: Navigate large pipelines easily

### User Experience
- **Responsive Design**: Works on desktop and tablet
- **Dashboard**: Manage multiple pipelines
- **Analytics**: View pipeline metrics and performance


##  Getting Started
### Installation

1. Install dependencies:

```
cd frontend
npm install
```

2. Configure environment:

Create .env file from template:

```
cp .env.template .env
```


Edit .env:
```
VITE_API_URL=http://localhost:8081
```
### Development

Start the development server:

```
npm run dev
```
